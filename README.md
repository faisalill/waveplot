# waveplot
waveplot is a VCD waveform generator for the terminal.

### Requirements
- cargo (Rust)

### Installation
```bash
cargo install waveplot
```

### Usage 
- help
```bash
waveplot --help
```

- version
```bash
waveplot --version
```

- Generate a waveform from a VCD file
```bash
waveplot <vcd_file> 
```
